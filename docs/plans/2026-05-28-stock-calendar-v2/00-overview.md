# 00 — 產品願景與功能總覽

## 產品定位

**股市行事曆 v2** 是一個給散戶 / 股市愛好者使用的「**個人預測筆記本**」，主要用途是社群娛樂。

- 使用者輸入股票（台股 / 美股）→ 在月曆上預測每日漲跌停或自訂價格 → 一鍵產生精美圖片 → 分享到外部社群（Threads / IG / LINE / Facebook）
- **不做站內社群**，所有互動發生在 App 之外
- 預測結果隔日可自動結算（後端有資料時）或手動回填，累積個人準度紀錄

## 目標用戶

- 台股 / 美股散戶
- 喜歡在社群（特別是 Threads、Dcard 股板）發梗圖、嘴砲預測的用戶
- 想記錄自己看法是否準確的人

## 核心功能（Phase 1 上架版本）

### 1. 股票管理
- 新增 / 移除追蹤股票（台股 4 碼代號、美股 ticker）
- 多支股票切換瀏覽
- 自動帶入股票名稱（透過後端 API）

### 2. 月曆預測輸入
- 點擊日期 → 彈出輸入面板
- 預測類型：
  - **漲停** / **跌停**（依市場規則：台股 ±10%，美股無漲跌停但保留為「強漲 / 強跌」標籤）
  - **自訂價格**（直接輸入數字）
  - **自訂漲跌幅 %**
  - **多 / 空看法**（純情緒標記，無數字）
- 備註欄（選填）

### 3. 自動結算（混合策略）
- 預設由 `stock.wisplu.com.tw` API 取得實際收盤價
- API 失敗（停電 / 斷網 / 維護）→ 提示「無法取得，請手動輸入收盤價」
- 結算結果：命中 / 偏差 %、染色顯示在日曆上

### 4. 多風格行事曆主題
- Phase 1 內建 3–5 個免費主題（淺色、暗色、財經紅綠、極簡、迷因風）
- 主題切換立即套用整月視覺
- Phase 2 開放更多付費主題

### 5. 出圖分享（多版型）
- **整月行事曆截圖**：完整月曆 + 預測
- **單日預測卡片**：聚焦某一天的預測，適合即時發文
- **月度準度報告卡片**：本月命中率、最神準預測、總漲跌幅統計
- 所有版型 → 存相簿 + 系統 share sheet（不上傳雲端）

### 6. 準度報告
- 個人歷史命中率
- 各股票準度排行
- 月 / 季統計圖表（fl_chart）

### 7. 帳號與資料
- 預設匿名登入，立刻可用
- 提示綁定 Apple（iOS）/ Google 以跨裝置同步
- 行事曆資料：本地 Hive 優先 + Firestore 同步

### 8. 通知
- 本地排程：收盤後提醒回填、結算結果通知
- 同時上傳 FCM token 到 Firestore，為 Phase 2 後端推播預留

### 9. 廣告
- AdMob banner（首頁底部）
- AdMob interstitial（出圖完成後可選擇性顯示，控制頻率）

## 非功能性需求

- **離線可用**：本地 Hive 為主資料源，飛航模式仍可記錄預測
- **降級設計**：股價 API 失敗不影響 App 使用
- **隱私**：圖片預設不上雲，僅本地產生

## 不做的事（明確排除）

- ❌ 站內社群（按讚、留言、追蹤）
- ❌ K 線圖即時報價
- ❌ 自動下單 / 券商整合
- ❌ 群組功能
- ❌ Phase 1 不做付費功能（訂閱 / IAP）
- ❌ Phase 1 不上傳分享圖片到雲端
